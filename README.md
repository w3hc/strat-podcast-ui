# Strat Podcast UI

## Install

```bash
pnpm install
```

## Run

Create a `.env` on the model of `.env.template`:

```sh
cp .env.template .env
```

Add your own keys in your `.env` file, then:

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Versions

- Node [v18.17.1](https://nodejs.org/uk/blog/release/v18.17.1/)
- PNPM [v8.7.5](https://pnpm.io/pnpm-vs-npm)
